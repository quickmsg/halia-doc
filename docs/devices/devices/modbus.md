# Modbus

### 设备
- **物理链路**：可选择以太网或串口。
- **编码方式**：可选择tcp或rtu over tcp。
- **指令发送间隔**：每条读或写指令之间的时间间隔。
- **重连时间**：说明此规则配置的作用，方便后续检查与管理。
- **host**: 目标设备的IP地址或域名地址。
- **端口**: 目标设备的端口。

### 源
- **从站**: modbus协议的slave值。
- **地址**: 点位的地址，从0开始。
- **字段**: 产生值的字段。
- **区域**: 可选线圈状态、离散输入状态、输入寄存器、保持寄存器。
- **类型**：值的类型，支持bool、uint8、int8、uint16、int16、uint32、int32、uint64、int64、float32、float64、string、bytes。
- **编码**：不同的类型对应不同的编码。
- **间隔**：轮询的间隔。

### 动作
- **从站**: modbus协议的slave值。
- **地址**: 点位的地址，从0开始。
- **区域**: 可选线圈状态、离散输入状态、输入寄存器、保持寄存器。
- **类型**：值的类型，支持bool、uint8、int8、uint16、int16、uint32、int32、uint64、int64、float32、float64、string、bytes。
- **编码**：不同的类型对应不同的编码。
